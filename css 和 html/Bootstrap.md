Bootsrap中的全局样式

Bootstrap提供了大量的全局样式，基本的HTML元素均可以通过class设置样式得到增强效果。

### Bootstrap中的排版 ——标题

标题(h1~h6 / .h1~h6)

副标题(small)

h1:36px;

h2:30px;

h3:24px;

h4:18px;

h5:14px;

h6:12px;

大小标题写法

```html
小标题写法
<h1>标题1<small>小标题</small></h1>
<h2>标题2</h2>
<h3>标题3</h3>
<h4>标题4</h4>
<h5>标题5</h5>
<h6>标题6</h6>
模拟标题
<span class="h1">标题1</span>
```
