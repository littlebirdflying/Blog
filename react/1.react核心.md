## 使用脚手架创建项目

1. 安装官方脚手架： `npm install -g create-react-app`
2. 创建项目：`create-react-app 项目名`
3. 启动项目：`npm start`

## 文件结构

### 文件结构一览

```markdown
my-app
├── README.md                 文档
├── node_modules  
├── package.json              npm 依赖
├── .gitignore
├── public                    静态资源
│   ├── favicon.ico
│   ├── index.html
│   └── manifest.json
└── src                       源码
    ├── App.css
    ├── App.js                  根组件
    ├── App.test.js
    ├── index.css             全局样式
    ├── index.js              入口文件
    ├── logo.svg
    └── serviceWorker.js      pwa支持
```

### cra项目真容

使用`npm run eject`弹出项目真容，会多出两个目录：

```markdown
├── config                    静态资源
│   ├── env.js 处理.env环境变量配置文件
│   ├── paths.js 提供各种路径
│   ├── webpack.config.js webpack配置文件
│   └── webapckDevServer.config.js 测试服务器配置文件
└── script 启动、打包和测试脚本                       
    ├── build.js 打包脚本
    ├── start.js 启动脚本
    └── test.js 测试脚本
```
