# HTTP协议原理+实践

## 课程导学

对课程整体进行介绍，并且说明HTTP协议的重要性，以及学习的必要性。

### 1-1 导学

http 很重要，提升自身价值，例如输入URL 打开网页、ajax 获取数据、img 标签加载图片都离不开 http

http 可以做很多事情，Cache-Control、缓存验证、更多有意义的头、深入到TCP

学习 http 的好处

- 对于后端，能够打造性能更好的 http 服务
- 对于前端，能够使用 http 的特性进行开发
- 前后端更好协作
